Project Description
------------------------------------
This project implements a full NAO choreography for the NAO Planning Challenge.
The choreography follows the required structure with:

• 6 mandatory positions 
• More than 5 intermediate movements
• A total duration within the 2-minute limit
• Fully executable in the NAO Simulator via Choregraphe
• A Python "Algorithm A" that generates the same sequence used in the final choreography

The algorithm (algorithm.py) generates the ordered sequence of movements used
between mandatory positions and ensures the logical planning requirement of the challenge.

The Choregraphe project contains the full choreography exactly as generated by Algorithm A.

------------------------------------
How to Run (Simulation)
------------------------------------
1. Open Choregraphe inside the VM.
2. Go to: File → Open Project
3. Select the folder: Choregraphe_Project/
4. Press the PLAY button to start the choreography.
5. Ensure that the NAO virtual robot is connected (default in the VM).

------------------------------------
Required Libraries / Environment
------------------------------------
• NAOqi (already pre-installed in the official VM)
• Python 2.7 (default in VM)
• Choregraphe Suite (pre-installed in VM)
No additional libraries are required.
